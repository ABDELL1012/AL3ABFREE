# al3abfree

موقع بسيط فيه ألعاب مجانية للتحميل أو اللعب أونلاين.

## المحتوى
- ألعاب فلاش قديمة
- ألعاب مجانية للتحميل
- روابط مباشرة بدون إعلانات مزعجة

## طريقة التثبيت
1. فك الضغط على الملف `al3abfree_site.zip`
2. شغل الملف `index.html` فالمتصفح
